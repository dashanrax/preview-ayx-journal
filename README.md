# preview-ayx-journal

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面是静态内容，以文件形式保存并托管于此，方便浏览和回溯。

仓库本身不针对任何特定域名或网站，仅作为 HTML 文件的集合和展示空间。

## 目录结构

```
.
├── README.md           # 本文件
├── index.html          # 仓库首页（可选）
├── pages/              # 存放 HTML 页面文件的目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 页面所需的资源文件（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：主要 HTML 页面存放位置。
- `assets/`：页面依赖的静态资源，按类型组织。

## 页面归档说明

此仓库中的 HTML 页面均为独立文件，每个页面可单独访问。页面内容可能包含文本、图片、样式和脚本，均为静态资源，不依赖后端服务。

页面内容可能不定期更新或替换，历史版本可通过 Git 提交记录追溯。
